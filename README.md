## Ataques DoS no Windows RDP (tela azul)

![](https://www.sandsteinwandern.de/wandern/wp-content/uploads/2019/11/giphy-hack2.gif)

RDP é a abreviação de Remote Desktop Control, sendo uma opção para controlar um sistema de computador remotamente.

O serviço RDP pode ser configurado por administradores de sistemas Windows para ser executado em TCP (geralmente na porta 3389) e/ou na porta UDP (3389).
